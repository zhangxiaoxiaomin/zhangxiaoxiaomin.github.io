UML系列01-背景介绍

发表于 2018-01-17 | 分类于 产品方法论

写在前面的话

为什么要学UML？

我的困境：

  1. 从事软件行业，对软件过程的理解似是而非。迫切需要对软件的生命周期有深刻的认识。
  2. 作为产品策划人，做产品喜欢“拍脑袋”，想到哪儿就是哪儿，用户需求与最终结果之间不具有可追溯性。
  3. 产品需求文档写了不少，但不能很清晰地界定文档中各种图的使用场景，例如：活动图的作用是什么？什么场景下使用？
  4. 做出来的产品逻辑漏洞百出，是否有方法减少？

工作中面临的上述问题，是驱使我决心系统学习UML的动力。

工具系列02-使用SecureCRT抓盒子log

发表于 2017-09-06 | 分类于 工具

准备工作

下载SecureCRT、tcpdump；安装SecureCRT；拷贝tcpdump至U盘中。

连接盒子

  1. 将U盘外接在盒子上；
  2. 打开SecureCRT->快速连接->协议选talent->输入盒子的IP->确定；
  3. 连接成功后页面中会提示输入用户名、密码；输入用户名（root）、密码（空）后，连接成功盒子会重启；按Ctrl+C截断输出。
