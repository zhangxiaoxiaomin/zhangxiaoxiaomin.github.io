工具系列01-使用adb工具安装apk以及相关操作

发表于 2017-08-09 | 分类于 工具

本文记录如何使用adb工具安装软件到android盒子。

前期准备

安装adb

下载并安装adb工具。下载地址：http://dl.pconline.com.cn/download/359035.html
安装完成后，打开cmd窗口，输入命令adb，可查看adb工具的详细信息，否则安装失败。

通过IP连接设备

  1. 查看盒子的IP地址，改为自动获取IP地址。
  2. adb connect X.X.X.X 盒子ip
    adb remount 意思是将设备改为可读可写;

安装apk

方式一：install命令安装

adb install C:\XX.apk 安装

异常：daemon not running. starting it now on port 5037 *
解决方案：http://blog.csdn.net/xishuluoye/article/details/17880663

若需要再次安装：
有以下两种方式：

  1. adb install -r C:\XX.apk /system/app （覆盖安装）
  2. adb uninstall 包名 （卸载后，再次安装）

方式二：push命令安装

将安装文件放到盒子的指定路径下即可完成安装。
依次输入以下命令：
1.adb shell
2.ls /system/app

3.mkdir /system/app/应用名称 //创建文件夹如果创建失败，可以先输入命令行 mount -o remount /system，再创建文件夹
4.adb push C:\XX.apk /system/app/XX/ 将要安装的应用push到盒子指定目录下
5.重启盒子
现在，可在我的应用里找到新安装的应用。（/(ㄒoㄒ)/~~

抓android盒子的log

adb logcat -v time ->E:\a.log //抓log

通过命令启动盒子里的应用

adb shell
am start -n 包名/类名

查看盒子目录的命令

adb shell
cd 文件路径
ls

其他操作命令

ctrl+c //退出

rm -f 文件名 //删除某个文件

adb shell tcpdump -n -s 0 -i eth0 -vv -w /data/data/xxx.pcap //抓网络包

adb disconnect ip //解绑设备

adb reboot //重启设备

adb shell pm clear 包名 //退出应用

华为手机开启日志，在手机拨号界面输入 * # * # 2846579 # * # * 进行设置

# adb # 安装apk
