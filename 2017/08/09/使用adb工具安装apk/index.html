工具系列01-使用adb工具安装apk

发表于 2017-08-09

本文记录如何使用adb工具安装软件到android盒子。

安装adb

下载并安装adb工具。下载地址：http://dl.pconline.com.cn/download/359035.html
安装完成后，打开cmd窗口，输入命令adb，可查看adb工具的详细信息，否则安装失败。

通过IP连接设备

  1. 查看盒子的IP地址，改为自动获取IP地址。
  2. adb connect X.X.X.X 盒子ip
    adb remount 意思是将设备改为可读可写;

安装apk

有两种方式：

  1. push
    查看盒子的目录/system/app：1.输入adb shell;2.输入ls /system/app
    发现指定目录下为文件夹，随便打开一个文件夹，可发现通常结构为application/application.apk

创建文件夹 mkdir /system/app/应用名称
如果创建失败，可以先输入命令行 mount -o remount /system，再创建文件夹
将要安装的应用push到盒子指定目录下 adb push C:\XX.apk /system/app/XX/
重启，可在我的应用里找到新安装的应用。（/(ㄒoㄒ)/~~ 如果找不到，只能通过命令行手动启动了，方法见下文）

  1. install
    adb install C:\XX.apk 安装

异常：daemon not running. starting it now on port 5037 *
解决方案：http://blog.csdn.net/xishuluoye/article/details/17880663

若需要再次安装：
有两种方式：

  1. adb install -r C:\XX.apk /system/app （覆盖安装）
  2. adb uninstall 包名 （卸载后，再次安装）

手动启动

adb shell
am start -n 包名/类名

查看盒子目录

adb shell
cd 路径
ls
可查看盒子指定路径下的文件夹

其他操作命令

ctrl+c 退出

rm -f 文件名 删除某个文件

adb logcat -v time ->E:\a.log 抓log

adb disconnect ip 解绑设备

adb reboot 重启设备
